The annotated data are used in "Renxian Zhang, Dehong Gao, and Wenjie Li, 2011, What Are Tweeters Doing: Recognizing Speech Acts in Twitter. In AAAI-11 Workshop on Analyzing Microtext." Please cite if you use them in your work.

Each line of the topic files is in the format of 
TWEET TEXT :: SPEECH ACT TYPE, 
where SPEECH ACT TYPE is one of the following:

sta: statement
com: comment
que: question
sug: suggestion
mis: miscellaneous

Note that the tweets are only randomly sampled collections and you should not assume any time order.
